Create Grid

  1. Four paragraphs are grid items
  2. Container is going to be grid
  3. Target container using container class and then set the display to grid
  4. grid-template-columns is creating column of fractional ratio of 1 to 2.
    • If first column had width of 1 then second column is going to be twice the width of first column
    • The default behavior for a grid container is to try and take up the full width but only have as much height as it allows to fit the content so we don't have to change the height.
  5. grid-template-rows is creating rows of fractional ratio of 1 to 1.
    • We get rows of equal heights
  6. gap gives distance of 10px between columns and rows.
    • Allows paragraphs to separate from each other without any sort of margin or padding.

Grid Sizing

Fixed Size Grid : Non-Responsive

  1. Fixed sized columns and rows are not responsive
    • pixels

    • rem : this is still going to be fixed because it is relative to root of HTML
  2. We can combine grid-template-columns and grid-template-rows into one property, which is just called grid template
    • Define rows first and then add a forward slash(/) and then define columns
    • To define a lot of grids we use this shorthand.

Auto Size Grid : Responsive

  1. auto keyword to define grid
    • It is going to give us a little bit of responsiveness.
    • Each row is going to try and fit to 100% of the horizontal available space.
    • 100% Width : The second columns is going to be as wide as it needs to in order to take up 100% of the width.
    • Fit Content : In case of row auto keyword is not going to behave the same way as columns and instead of this it tries to fit to the content.
  2. Grid Sizing Demo

Fractional Size Grid

  1. Defines different areas of layout with ratios for sizes.
  2. First row is going to be half of the second row.
  3. First column is going to be half of the second column.
  4. fr : Fractional unit which defines ratios.
    • It's going to shrink and grow depeneding on the width available for the horizontal and also the content size of the vertical.

Grid Placement

Introduction

  1. Layout items in the grid and place them in the areas that we want.
  2. Great layout to use CSS grid.
  3. Terms often used in CSS Grid
    • Container : div called Grid container that contains all of the items.
    • Items inside grid : divs called as grid items
    • Rows and Columns : We create and size our grid through rows and columns which are collectively knowns as Tracks.
      • Row Tracks : Created and sized by grid-template-rows.
      • Column Tracks : Created and sized by grid-template-columns.
    • Grid Cell : Within the intersection of tracks, we create small units which are called a grid cell.
      • Grid cell is the smallest unit in a grid and they can be different sizes.
      • Multiple cells can be us ed to create grid items.
      • Here, grid item is positioned across two of our grid cells.Two items are positioned vertically and one item is positioned horizontally.
    • Grid Lines : Horizontal and vertical lines that separate tracks.
      • Grid lines are controlled by using gap property to specify it's height and width.
      • But we can't change its color.
  4. With all of things combined together in a grid, we've got our lines,tracks,cells,container and items.
    • These all things combined together that allows us to create these layouts using our grid.
